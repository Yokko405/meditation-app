# 瞑想タイマー - Inner Peace

美しいグラデーション背景とシンプルなUIを持つ瞑想タイマーアプリです。

## 特徴

- 🎨 美しいグラデーション背景（アニメーション付き）
- ⏱️ カスタマイズ可能なタイマー設定
- 🎵 自然音の再生機能
- 📱 レスポンシブデザイン（モバイル対応）
- 🌐 ブラウザで直接動作（インストール不要）

## 使い方

### GitHub Pagesで公開中

以下のURLからアクセスできます：

**https://yokko405.github.io/meditation-app/**

### ローカルで実行する場合

1. リポジトリをクローン
```bash
git clone https://github.com/Yokko405/meditation-app.git
cd meditation-app
```

2. ブラウザで `index.html` を開く
```bash
open index.html
```

または、ローカルサーバーを起動：
```bash
# Python 3の場合
python3 -m http.server 8000

# Node.jsの場合
npx http-server
```

ブラウザで `http://localhost:8000` にアクセスしてください。

## ファイル構成

- `index.html` - メインの瞑想タイマーアプリ
- `languages.html` - プログラミング言語一覧ページ
- `languages.json` - 言語データ（JSON形式）
- `start.sh` / `start.command` - ローカルサーバー起動スクリプト
